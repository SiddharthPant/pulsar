# Pulsar
This is a template project. Despite being that its very production-esque and its my intention to do so, because the idea is just that. Create a production ready template using Rust and Datastar. A stack that can last for a decade and beyond. A toyota car that keeps running.

## The Decade Stack
### Backend
- Axum
- Askama
- NATS
### Frontend
- Datastar
- HTML/CSS
- Web components
- Some custom JS
